PayPay 応援QRの置き場所

このフォルダに、PayPayの「受け取り用QRコード（MyCode）」の画像を
次のファイル名で置いてください：

    paypay-qr.png

WEEKY の 設定 →「💛 応援」に自動で表示されます。
ファイルが無い間は「QR画像を準備中」のプレースホルダーが出ます。

メモ:
- PayPayのQRは定期的に差し替えが必要なので、新しい画像で同じ名前のまま上書きしてください。
- 画像を差し替えても古いものが表示される場合は、アプリのバージョン
  （APP_VERSION / service-worker.js の CACHE_NAME / index.html の ?v=）を
  上げるとキャッシュが更新されます。
- 公開（投げ銭リンク設置）の前に、勤務先・教育委員会への相談を忘れずに。
